# Simulador de Mercado Tipo Uber

Este repositorio contiene un simulador simplificado de un mercado tipo Uber,
modelando la interacción entre oferta (conductores), demanda (usuarios) y
pricing dinámico (surge).

## Cómo correrlo

La aplicación está diseñada para ejecutarse directamente en Streamlit Cloud.

### Parámetros principales
- Número de usuarios
- Número de conductores
- Precio base
- Sensibilidad del surge pricing
- Horizonte de simulación

## Output
- KPIs ejecutivos
- Gráficas de evolución temporal
- Tabla detallada por tick
- Exportación a CSV
